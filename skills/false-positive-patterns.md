# False Positive Patterns

Track patterns that caused junk alerts.

Initial patterns:
- People asking general questions about promos without an actual offer.
- Expired promo discussions.
- Jokes containing promo keywords.
- Flexing previous discounts with no current action.
- Vague "masih bisa?" messages without original context.
- Repeated duplicate confirmations after an alert already fired.
- Messages that are just platform names without a deal ("tokped", "shopee" alone).
- Price discussions without actionable discount info.
- "Udah abis" / "gak work" responses that indicate expiry.

Update this file from feedback.
