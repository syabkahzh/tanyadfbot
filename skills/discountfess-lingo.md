# Discountfess Lingo

Maintain Indonesian discount/promo slang and meanings.

Seed terms:

- cb = cashback
- vc = voucher
- cap = maximum cashback/discount
- min / minbel = minimum spend
- kuota = limited redemption quota
- targeted / selected = not available to all users
- work / masih bisa = users report promo still active
- abis / kuota habis = likely expired or no longer useful
- stack / distack = can/cannot combine promos
- ep = error price
- qris = QRIS payment
- spay = ShopeePay
- spaylater = ShopeePay Later
- gosend/gofood/grabfood = service/platform context
- lzd = Lazada
- tokped / toped = Tokopedia
- gf = GoFood
- idm = Indomaret
- klg = Kelapa Gading (mall area)
- kesbek / keshbek / cashbek / casback = cashback misspellings
- vocer / voucer = voucher misspellings
- ongkir = shipping fee (gratis ongkir = free shipping)

Update this file when feedback reveals new slang or changed meanings.
